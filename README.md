# WordFlow

A speed-reading application that helps to read PDFs faster and more efficiently.

## Features

- **Speed Reader**: Adjustable WPM (60-600) with word-by-word display.
- **Activity Tracking**: Visual timeline of your daily reading habits.
- **Responsive Design**: Clean, distracting-free interface.

## Quick Start

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Run application
python run.py
```

Open [http://localhost:5000](http://localhost:5000) in your browser.
